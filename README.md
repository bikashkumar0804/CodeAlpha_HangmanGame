# CodeAlpha Hangman Game

A professional text-based Hangman Game developed using Python.

## Features
- Difficulty Levels
- Score System
- Hint Feature
- ASCII Graphics
- Leaderboard File
- Input Validation
- Replay Option

## Technologies Used
- Python
- File Handling
- Loops
- Functions
- Lists & Dictionaries

## How to Run
```bash
python hangman.py

```
